# AIGC 创作平台 (AIStudio) Mock API

模拟 AIGC 创作平台与第三方 AI API 的完整调用链路，专为接口集成测试设计。

## 快速启动

```bash
cd aistudio-mock-api
npm install
npm start
# http://localhost:3000
```

## 架构

```
┌─────────────────────────────────────────────────────┐
│                  AIGC 创作平台                        │
├─────────┬─────────┬─────────┬──────────┬────────────┤
│  认证    │  模型    │  项目    │  任务     │  对话/用量  │
│  5 APIs │  3 APIs │  4 APIs │  6 APIs  │  4 APIs    │
└─────────┴─────────┴─────────┴──────────┴────────────┘
         共 22 个接口 | 6 大模块
```

## 核心链路

```
登录鉴权 → 选择模型 → 提交任务 → 轮询状态 → 获取结果
  1.2        2.1/2.2    4.1/4.2     4.3       4.3 result
```
